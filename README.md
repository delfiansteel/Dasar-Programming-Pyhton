Ketikkan fungsi dan string sebagai berikut<p>

```
##Input##
name = "Delfian"
kelas = "A1"
umur = "18 tahun"
##Output##
print ("Nama saya "+name,"saya dari kelas "+kelas,"dan umur saya "+umur,".")
```
Maka hasilnya akan seperti ini<p>
```
Nama saya Delfian saya dari kelas A1 dan umur saya 18 tahun .
```
